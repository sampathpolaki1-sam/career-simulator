# CareerForge - AI Career Decision Simulator 2026

## Overview
CareerForge is an advanced AI-powered career decision simulator that helps users explore and analyze different career paths. Built with modern web technologies, it features an intuitive interface with real-time career recommendations.

## Features
- **AI-Powered Career Recommendations**: Get personalized career suggestions based on your profile
- **Interactive Simulation**: Experience different career scenarios before making decisions
- **Real-Time Analytics**: Track and analyze your career progression
- **Beautiful UI**: Modern, dark-themed interface with smooth animations
- **Secure Authentication**: Safe login and signup with encrypted credentials

## Technologies
- HTML5
- TailwindCSS (via CDN)
- JavaScript (Vanilla)
- Font Awesome Icons
- Modern CSS with Glassmorphism effects

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for CDN resources)

### Installation
1. Clone the repository:
```bash
git clone https://github.com/sampathpolaki1-sam/career-simulator.git
cd career-simulator
```

2. Open the application:
```bash
# Simply open featuredesider.html in your browser
# Or use a local server for better performance
python -m http.server 8000
# Then visit http://localhost:8000/featuredesider.html
```
